Secuencia de tareas y actividades

Tareas y actividades

Para la realización de este proyecto se llevará a cabo un conjunto de actividades y tareas organizadas de forma progresiva y secuenciada, con el objetivo de facilitar la consecución del producto final: la elaboración de un informativo meteorológico. Estas tareas están diseñadas para guiar al alumnado a lo largo de las distintas fases del proceso, desde la investigación y recopilación de información hasta la planificación, elaboración del guion y producción del material audiovisual. De este modo, se favorece un aprendizaje estructurado que permite integrar conocimientos, desarrollar competencias comunicativas y digitales, y terminar en la creación de un producto final coherente y funcional adecuado a las características y contexto del alumnado.
